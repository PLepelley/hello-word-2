Première étape du tuto Git :
Créer un dossier : mkdir "name_file"
Activer git pour un répertoire : git init
Vérifier le statut : git status
Pour ajouter un fichier que l'on vient de créer : git add 'readme.txt'
Pour soummetre ces modifications : git commit -m "Création du fichier readme"
Pour afficher les différence : git diff
